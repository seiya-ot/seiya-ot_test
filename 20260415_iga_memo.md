# IID

## 概念

```
[dev,ope]プロダクトビジョン（事業・プロダクトの大きな方向性）
        ↓
[dev,ope]エピックバックログ（大きな塊・テーマ）
    アプリケーションログインポリシー機能を実装したい。
        ↓
[dev]プロダクトバックログ（やること一覧・優先順位付き）
    ログインポリシーの編集画面を実装する。
    ログインポリシーの追加画面を実装する。
        ↓
[dev,ope]スプリントバックログ（今スプリントで実際にやるタスク）
    Unitテストを書く。
    GET /loginpolicies APIを実装する。
    int 動作確認を実施する。
    レビューする。
```

- プロダクトビジョン（事業・プロダクトの大きな方向性）

- エピックバックログ（大きな塊・テーマ）: komainu/backlog リポジトリのissue
  - https://gh.iiji.jp/komainu/backlog/issues
- プロダクトバックログ（やること一覧・優先順位付き）: komainu/product-backlog リポジトリのissue
  - https://gh.iiji.jp/komainu/product-backlog/issues

- スプリントバックログ（今スプリントで実際にやるタスク）: komainu/sprint-backlog リポジトリのissue
  - https://gh.iiji.jp/orgs/komainu/projects/5/views/1
  - かんばん管理
